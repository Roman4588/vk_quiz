# Структура интерфейса для переноса в Miro/Figma

## Экран 1. Авторизация

- Заголовок продукта.
- Краткое описание: квизы в реальном времени.
- Переключатель «Вход / Регистрация».
- Поля email и пароль.
- При регистрации: имя и роль.

## Экран 2. Кабинет организатора

- Список созданных квизов.
- Форма создания квиза.
- Кнопка запуска комнаты.
- Блок с историей проведённых квизов.

## Экран 3. Создание квиза

- Название.
- Категория.
- Правила.
- Время на вопрос.
- Список вопросов.
- Для каждого вопроса: тип, текст, URL изображения, варианты ответа, правильные ответы.

## Экран 4. Комната организатора

- Крупный код комнаты.
- Текущий вопрос.
- Кнопки управления: начать, закрыть вопрос, следующий вопрос, завершить.
- Список участников.
- Лидерборд.

## Экран 5. Кабинет участника

- Поле ввода кода комнаты.
- Информация о текущей комнате.
- Блок вопроса.
- Варианты ответа.
- Кнопка отправки ответа.
- Лидерборд.
- История участия.
